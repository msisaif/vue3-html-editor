# Vue3 HTML Editor

Vue3 + Vue Router + Pinia + Tailwind CSS + vite

## Clone from git

```sh
git clone https://github.com/msisaif/vue3-html-editor.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
